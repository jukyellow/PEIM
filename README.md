# PEIM
> 정보관리기술사(Professional Engineer Information Management) 자격증 도전  
<br>

- 학습기간: 2015년 11월 ~ 2016년 7월 (9개월)  
- 학습토픽  
1. 소프트웨어공학
2. 데이터베이스
3. 네트워크
4. CA_OS
5. 정보보안
6. 경영정보
7. 디지털 신기술
8. 알고리즘
<br>

- 자격증 전형과정
 1. 필기시험: 서술형 필기, cut off 60점(상대평가 작용, 59점에 100명씩 포진..)  
 2. 실기전형: 기술사/교수 면접  
<br>

- 답안방식 및 문제갯수/시험시간
 1. 답안방식 : 서술형, 필기시험
 2. 문제갯수/시험시간
  > 1교시 : 13문제 중 10문제 선택, 시험시간 100분  
  > 2~4교시 : 6문제 중 4문제 선택, 시험시간 100분
